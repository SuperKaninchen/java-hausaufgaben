## Folder/File Structure
The folders and files are structured like this:

* YEAR MONTH (eg. 2019_Dezember)
  * PAGE (eg. AB_8)
    * TASK (eg. Aufgabe1)
    * SOLUTION (eg. Aufgabe1_loesung)
